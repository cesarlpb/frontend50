# CSS 3 - Navbar

```
Se te proporciona un archivo HTML para la barra de navegación superior de un sitio web. La barra de navegación consta de dos listas no ordenadas, que contienen tres y dos elementos de lista, respectivamente.

Estiliza esta barra de navegación de la siguiente manera:

-  La barra de navegación debe ser una barra horizontal que abarque todo el ancho de su elemento padre; todos los elementos de lista deben mostrarse en una sola fila.
- La primera lista no ordenada (que contiene Product, Testimonials y Pricing) debe aparecer en el extremo izquierdo de la barra de navegación. La segunda lista no ordenada (que contiene Team y Contact Us) debe aparecer en el extremo derecho de la barra de navegación.
- La barra de navegación debe tener un color de fondo de #6e072d  y un color de fuente blanco.
Las listas no ordenadas no deben tener margen ni relleno. Sin embargo, los elementos de lista individuales deben tener un relleno vertical igual a su tamaño de fuente y un relleno horizontal igual a su tamaño de fuente multiplicado por 1.5; el tamaño de fuente debe ser de 18px.
- Al pasar el cursor sobre un elemento de lista, su color de fondo debe cambiar a #b90e4d  y el cursor del mouse debe cambiar a un puntero.
- Debe haber un borde sólido, blanco, de 2px que separe los elementos de lista. Sin embargo, este borde no debe aparecer en el lado derecho de Pricing o en el lado izquierdo de Team.
Los elementos individuales de lista no deben tener viñetas junto a ellos.

Tu entrega será evaluada en función de cuán similar sea tu resultado en el navegador al resultado esperado; debe ser casi idéntico.
```

## HTML (no modificar)
```html
  <nav>
    <ul>
      <li>Product</li>
      <li>Testimonial</li>
      <li>Pricing</li>
    </ul>
    <ul>
      <li>Team</li>
      <li>Contact Us</li>
    </ul>
  </nav>
```
